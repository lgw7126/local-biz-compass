## 기획 의도

동네 카페·식당 사장님들은 매일 아침 두 가지를 고민합니다.

- **"오늘 재료를 얼마나 준비하지?"**
- **"내 주변에 경쟁 가게가 얼마나 있지?"**

기존 상권 분석 서비스는 복잡한 차트와 통계 용어로 가득 차 있어 바쁜 사장님들이 일상적으로 쓰기 어렵습니다.

**동네상권 나침반**은 단 3초 만에 핵심 정보를 파악할 수 있도록 설계했습니다.

- 📍 내 동네 경쟁 업소 수 + 가게 이름/주소 확인
- 🌧️ 내일 날씨 및 강수확률 실시간 연동
- 💡 날씨에 따른 맞춤 재료 발주 조언 (3단계 신호등)
- 📅 1주일 날씨 예보 그래프

모든 데이터는 **공공데이터(무료)** 를 활용하여 추가 비용이 없습니다.

---

## 사용법

### 1. 사전 준비

```bash
# 저장소 클론
git clone https://github.com/lgw7126/local-biz-compass.git
cd local-biz-compass
```

**필요한 파일 2개를 직접 준비해야 합니다:**

| 파일 | 출처 | 방법 |
|------|------|------|
| `mapo_stores.csv` | 소상공인시장진흥공단 상가정보 | 공공데이터포털에서 서울 CSV 다운로드 후 마포구 필터링 |
| `.env` | 기상청 단기예보 API 키 | 공공데이터포털에서 무료 신청 |

`.env` 파일 형식:
```
WEATHER_API_KEY=여기에_발급받은_인증키_입력
```

### 2. 실행

```bash
uv run --with streamlit --with pandas --with requests --with python-dotenv \
  python -m streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속

### 3. 화면 사용법

1. **왼쪽 사이드바**에서 법정동과 업종을 선택합니다
2. **경쟁 업소 수** 아래 "가게 목록 보기"를 클릭하면 상호명과 주소가 펼쳐집니다
3. **조언 카드** 색상으로 오늘의 상황을 즉시 파악합니다
   - 🟢 초록: 평소대로 운영
   - 🟠 주황: 재료 90% 준비
   - 🔴 빨강: 발주량 줄이기 권장
4. **하단 그래프**에서 1주일 기온 변화와 강수확률을 확인합니다

## 기술 스택

- **Python** + **Streamlit** — 웹 대시보드
- **Pandas** — 상가 데이터 필터링
- **소상공인시장진흥공단 상가(상권)정보** — 경쟁 업소 데이터
- **기상청 단기예보 / 중기예보 API** — 날씨 데이터
